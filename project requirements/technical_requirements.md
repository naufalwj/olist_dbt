# 1. Olist Marketplace Analytics

## Technical Project Requirements (to do)
### Medallion Layers
#### Bronze Layer (raw)
- **One Delta table** per source CSV
- Column names preserved as-is from source
  - lower_snake_case
- Data types cast **from string to appropriate native types** (DATE, TIMESTAMP, FLOAT, INT)
- **No filtering**, **deduplication**, or **business logic**

#### Silver Layer #1 (staging)
- Translate product categories **from Portuguese to English** (join translation table)
- **Apply dbt not-null, unique, and relationship tests** to validate referential integrity

#### Silver Layer #2 (intermediate)
- **Handle known null patterns**. Flag orders with no delivery timestamp, mark cancelled orders
- **Derive basic flags**: is_delivered, is_late, has_review
- **Derive basic measures**: delay_days, total_revenue, total_freight
- **Derive basic aggregations**: total_orders, total_items, total_customers, total_sellers
- **Derive basic dimensions**: dim_customers, dim_sellers, dim_products, dim_date
- **Apply dbt not-null, unique, and relationship tests** to validate referential integrity
